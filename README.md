ОСНОВНОЕ:
• Меню 'Шифровки' - выбор алгоритма
• Верхнее поле - исходный текст
• Нижнее поле - результат
• Поле 'Ключ' - ключ шифрования

ШАГИ:
1. Выбрать алгоритм
2. Ввести текст и ключ
3. Нажать 'Шифровать'/'Расшифровать'

КЛЮЧИ:
• Цезарь: число
• Виженер: слово
• Атбаш: не нужен
• Сохраняйте ключи!"
